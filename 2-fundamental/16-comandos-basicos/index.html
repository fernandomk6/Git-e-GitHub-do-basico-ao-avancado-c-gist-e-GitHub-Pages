Coamandos Básicos

git init

Inicia um repositorio novo

git status

Verifica o estado dos arquivos

git add .

Adiciona todos arquivos para dentro da "visão" do git

git commit -a -m "O que foi alterado"

Salva um historico e uma mensagem

git push

Envia os arquivos locais, para o repositorio online

git pull

Atualiza o repositorio local com base no online

git checkout

Retorna o arquivo ao seu estado original
