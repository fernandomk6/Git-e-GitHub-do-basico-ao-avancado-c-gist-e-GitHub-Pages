Branches

Branches é a forma como o git separa as versões do projeto. Essas versões podem ser unidas a branch principal posteriormente

Visualizando todas as branchs

git branch é o comando usado para visualizar todas as branchs no terminal. Isso pode ser feito no gitHub também.

Criando uma branch

git branch nome-da-branch é o comando usado para criar branch branchs no terminal.

Deletando uma branch

git branch -d nome-da-branch é o comando usado para deletar branch branchs no terminal.

Trocando de branch

git checkout nome-da-branch é o comando usado para trocar de branch .

Esse é legal :D

Unindo branch

git merge nome-da-branch é o comando usado para unir branch. O branch atual sera unido com o branch informado no comando

Esse é legal :D

Git Stash

git stash é o comando usado para jogar as modificações atuais na lixeira

git stash list pode ser usado para verificar a "lixeira"

git stash apply id-do-stash pode ser usado para recuperar o codigo do stash
